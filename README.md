# ServLink

ServLink is a service management platform that connects users with service providers. It allows users to book services, track requests, and manage service updates through a simple and user-friendly interface.

## Live Demo
🔗 Website: https://servlink.netlify.app/

## Features
- User Registration & Login  
- Service Booking System  
- Request Tracking  
- Service Provider Management  
- Responsive UI  
- Easy Navigation  

## Tech Stack
- Frontend: HTML, CSS, JavaScript / Next.js  
- Backend: Python Flask  
- Database: SQLite / MySQL  

## Installation

1. Clone the repository:
```bash
git clone <repository-link>
```

2. Go to project folder:
```bash
cd servlink
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run backend:
```bash
python app.py
```

5. Open the frontend in browser.

## Project Structure
- Frontend for UI  
- Backend APIs using Flask  
- Database for storing user and service data
